Rock, Paper, Scissors

View Full Website

HTML & CSS

The website has a title saying rock paper and scissors. There is then a play button under it. If you click the play button the game will show up. The game screen has scores, the player hand and computer hand, and three buttons to allow the player which option they want to choose.

Javascript

To make the images change the player hand and computer hand were targeted with querySelector. Then an array is made that holds the different options for the images. This array is for the computer to pick a random option. Then there is a complex system of if statements that compare the hands to see who wins. Then the winner gets a new score and the text above the hands changes.
