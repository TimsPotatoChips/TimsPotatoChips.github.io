    The process I take to build my websites

    1

    Plan

    I plan the colors and basic design of the website. I process the website purpose and plan out it's layout, color, and font to match the purpose.

    2

    Design

    I then design the website using the colors, font, and layout with Figma. I add small interactions to calculate what kind of animations I should use.

    3

    Create

    I then go to either Webflow or Visual Studio Code to make the website I planned out. Then I add animations and make the website completley mobile responsive.
