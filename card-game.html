War Card Game

View Full Website

HTML & CSS

The website has a landing animation of a sliding background and the game mat appears. In the middle is the discard mat and on each side there are the player mats. Each mat has a score, card, and draw button. And at the bottom of the discard and player mats there is text that tells you the winner or wether the match was a tie.

Javascript & GSAP

The landing animations were made with green stock gsap animations. In the main javscript it targets the discard pile, the draw buttons, the match teller text, scores, and card images. It then makes an array of cards by using nested for each loops. It then uses a shuffle function that passes a deck, number of cards, and the deck it outputs the cards. Then it splits the deck and puts them in the a playerOneDeck and playerTwoDeck array. From each player deck it then draws a card and compares each when both draw buttons are clicked. It then changes the image based on the cards drawn. When it determines the winner the score changes and the winner gets the cards from the opponent. But if it's a tie it prints the images into the discard. Then when the next player wins it gets the cards in the discard and the images are removed.
